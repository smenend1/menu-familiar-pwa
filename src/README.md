# 🍽️ PWA Organizador de Menús Familiares

Aplicación Web Progresiva diseñada para organizar menús de 2 adultos y 2 niños (5 y 2 años) mediante menús desplegables adaptados al tacto en pantallas móviles.

## 🚀 Cómo subirlo a GitHub y desplegarlo gratis:

1. Crea un repositorio vacío en tu cuenta de GitHub llamado `menu-familiar-pwa`.
2. En la terminal de tu ordenador, dentro de la carpeta del proyecto, escribe:
   ```bash
   git init
   git add .
   git commit -m "Instalación inicial de la PWA familiar"
   git branch -M main
   git remote add origin TU_URL_DE_GITHUB_AQUÍ
   git push -u origin main
